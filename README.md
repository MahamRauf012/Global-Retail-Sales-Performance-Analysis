# Global Retail Sales Performance Analysis

## Overview
This is an independent data analytics project analyzing global retail transaction data to identify revenue drivers, sales trends, and top-performing markets, products, and customers.

The project demonstrates an end-to-end analytics workflow using Python, SQL, and Tableau.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Tableau
- Excel

## Project Structure
- **Data:** Raw dataset and data documentation
- **Notebooks:** Python data cleaning and analysis
- **SQL:** SQL queries used for validation and aggregation
- **Tableau:** Interactive dashboard (linked via Tableau Public)

## Key Insights
- Revenue is concentrated in a small number of countries.
- Sales increase significantly during later months of the year.
- A limited set of products and customers drives the majority of revenue.
